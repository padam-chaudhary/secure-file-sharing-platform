# Secure File Sharing Platform

Backend project built with **Laravel 11** focusing on real-world backend development and security best practices.

## Features
- Secure file upload & download
- Authentication & authorization
- Private and shareable file access
- Clean MVC architecture

## Tech Stack
- PHP 8.2+
- Laravel 11
- MySQL
- Git & GitHub

## Purpose
This project is being built as part of internship-level backend training with a focus on production-ready code and interview preparation.
